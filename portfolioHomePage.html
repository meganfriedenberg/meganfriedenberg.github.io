Custom Game Engine

For USC's undergrad game engines course, ITP485, I created a custom game engine using DirectX11, hlsl shaders, multi-threading, and developed a pipeline to load in assets from Blender. The engine also features 3D collision detections, skeletal animation programming, a camera with 6 degrees of freedom, normal mapping, the usage of multiple render targets, mipmapping, and billboarding. Shaders written for it include Valve's Half Lambert shader, Phong, diffuse, specular, a toon shader, and a skinned one to allow skeletons to be textured. You can check the code for it here.

Robot Rock

Created an automatic drumming bot with 4 other Makers. Developed a React UI to control the solenoids being powered by an Arduino Mega. Using signal processing in Python with SciPy and PyAudio, DrumBot can beat-detect surrounding instruments in real-time. TensorFlow's GrooVAE model are inputted the listened melody which allows DrumBot to play a beat to it. By developing a multi-threaded Python core, DrumBot can also translate into sheet music its performed beat in near constant time
